基於 Streamlit 開發的企業資源管理 (ERM) 工具，具備自動化案件評分、ROI（投資報酬率）分析儀表板及動態人力規劃模組。開發中，v3.1.1目前支援多檔案 Excel 數據輸入。
</br>
</br>
本工具適用對象：企業 RDM、有人力分配議題、專案成本效益議題之部門
